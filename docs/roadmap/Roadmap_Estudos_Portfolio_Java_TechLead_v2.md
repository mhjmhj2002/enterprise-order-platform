# Roadmap de Estudos Orientado a Portfolio

## Missao

Construir um portfolio de nivel Java Senior / Tech Lead, eliminando lacunas atraves de projetos reais e simulando o funcionamento de um time de engenharia.

## Ordem de evolucao

1. Sprint 0 - Fundacao arquitetural
2. Sprint 1 - Microsservicos REST
3. Sprint 2 - Escalabilidade de pedidos com eventos
4. Sprint 3 - Resiliencia
5. Sprint 4 - Performance e cache
6. Sprint 5 - Auditoria e historico
7. Sprint 6 - Comparacao com RabbitMQ
8. Sprint 7 - Observabilidade
9. Sprint 8 - Kubernetes
10. Sprint 9 - System Design e documentacao final
