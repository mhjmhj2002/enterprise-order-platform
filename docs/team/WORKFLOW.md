# Workflow

## Definition of Ready

A Story so pode entrar em desenvolvimento quando:

- possuir objetivo claro;
- possuir owner definido;
- possuir reviewers definidos;
- possuir criterios de aceite testaveis;
- possuir dependencias identificadas;
- possuir documentacao de negocio suficiente;
- nao possuir questoes em aberto bloqueantes.

## Definition of Done

A Story so e concluida quando:

- entregaveis foram produzidos;
- criterios de aceite foram atendidos;
- documentacao foi atualizada;
- PR foi revisado;
- item foi movido para Done no Project Board;
- CHANGELOG e PROJECT_HISTORY foram atualizados quando aplicavel.

## Document Lifecycle

Draft
↓
In Review
↓
Approved
↓
Baseline
↓
Deprecated

- **Draft:** documento em construcao inicial, ainda sujeito a alteracoes amplas.
- **In Review:** documento pronto para revisao formal de stakeholders definidos.
- **Approved:** documento validado pelos revisores responsaveis.
- **Baseline:** versao oficial de referencia para orientar stories e decisoes futuras.
- **Deprecated:** documento substituido ou sem validade para novas decisoes.
