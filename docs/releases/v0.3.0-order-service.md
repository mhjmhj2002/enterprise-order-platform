# v0.3.0-order-service

Release de conclusao da Story-009, validada pelo Quality Engineer em 10 de julho de 2026.

## Features

- Bootstrap completo do Order Service.
- Integracao REST com o Inventory Service.
- `PaymentFakeAdapter` configuravel.
- Casos de uso do ciclo de vida de pedidos.
- Testes unitarios e de integracao.
- Collection Postman autocontida.
- ADR-006 e documentacao arquitetural atualizada.

## Improvements

- Compensacao de estoque em falhas de reserva e pagamento.
- Health checks para Order e Inventory.
- Respostas `400` sanitizadas para JSON malformado e UUID invalido.
- Correlation ID aceito e propagado entre Order e Inventory.
